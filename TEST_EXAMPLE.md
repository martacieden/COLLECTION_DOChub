# Приклад тесту (без сценарію)

### Task X.X: [Назва завдання]

**Що відстежувати:**
- Чи знаходить користувач [елемент/функцію]?
- Чи розуміє [концепцію/термін]?
- Чи очікував він [поведінку/результат]?

**Фрази модератора:**
- ❌ "Натисніть кнопку X" → ✅ "Як би ви зробили [дію]?"
- ❌ "Це означає Y" → ✅ "Що ви думаєте означає [термін]?"
- ❌ "Ось правильне рішення" → ✅ "Що ви очікуєте побачити?"

**Pass/Fail notes:**
```
Success: [ ] Користувач виконав завдання без допомоги
         [ ] Розуміє [концепцію]
         [ ] [Конкретна метрика успіху]
         Time: _____ хвилин
         
Failure: [ ] Не знайшов [елемент]
         [ ] Запитав "[питання, що показує незрозумілість]"
         [ ] [Конкретна поведінка, що показує проблему]
         Confusion moments: _____
         Quotes: "_____"
```
